# Spolier

Спойлер можно использовать без каких либо стилей, а для оформление кнопки брать соответствующие класс (например `btn-primary-border`). 
Тут имеет смысл добавить дата-аттрибуты, меняющие текст при открытии/закрытии спойлера:
- data-is-toggle-text="true"
- data-text-open="Close"
- data-text-close="Read more"

### Пример
<Preview>
    <Story name="standard" height="auto">
        {() => {
            setTimeout(() => {
                App.initComponent(Spoiler, '.js-spoiler');
            }, 1);
                
            {/* }); */}
            return `
<p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Mollitia nostrum recusandae aliquid molestias nobis nam culpa est doloribus quia. Enim eius dignissimos sapiente nostrum quos porro, officia itaque autem quisquam deserunt iusto vel facilis.</p>
<div class="js-spoiler" data-is-toggle-text="true" data-text-open="Close" data-text-close="Read more">
    <div data-spoiler-content>

    </div>
    <button class="btn-primary-border" data-spoiler-btn>toggle btn</button>
</div>`; 
        }}
