# Buttons

With `MDX` we can define a story for `Checkbox` right in the middle of our
markdown documentation.


<Story name="default" height="100px">
    {() => {
        const textButton = text('text', 'I\'m button');
        const classList = {
            'btn-primary': 'btn-primary',
            'btn-secondary': 'btn-secondary',
            'btn-danger': 'btn-danger',
            'btn-warn': 'btn-warn',
            'btn-success': 'btn-success',
        }
        let html = '';
        Object.keys(classList).forEach((classname) => {
            html += `<button class="${classname}">${classname}</button>`;
        });
        /* const className = select('classname', classList, 'btn-primary'); */
        /* return `<button class="${className}">${textButton}</button>`; */
        return html;
    }}
</Story>
